# 🎭 Who Am I? – Party & Office Icebreaker Game

A fun and interactive web application that randomly assigns a word to each player, recreating the classic **"Who Am I?"** party game often played during office events, team-building activities, and social gatherings.

---

## 🎮 Overview

In the traditional "Who Am I?" game, each participant receives a hidden identity (person, object, animal, profession, etc.) placed on their forehead. Players then ask yes/no questions to others in an attempt to guess their assigned word.

This application automates the word generation process, making the game quick, fair, and easy to organize.

---

## ✨ Features

- 🎲 Random word generation
- 👥 Multi-player friendly
- 🏷️ Sticker-ready word display
- ⚡ Instant assignment without repetition
- 🎉 Simple and intuitive interface
- 📱 Works on desktop and mobile browsers

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript

---

## 🎯 How to Play

1. Open the application.
2. Generate a random word for each player.
3. Stick the displayed word on the player's forehead (without letting them see it).
4. Players ask only **Yes/No** questions.
5. The first player to correctly guess their word wins!
